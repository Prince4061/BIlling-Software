

# 🧠 🎯 **Project Goal**

👉 Offline Billing Software (Desktop App)

### Features:

* Data entry (customer + items)
* PDF bill generate (same design)
* Excel me data save
* Multi-item support
* Auto bill number
* Local use (no internet)

---

# 🏗️ **1. Overall Architecture (Simple View)**

```
[ UI (Tkinter) ]
        ↓
[ Business Logic ]
        ↓
[ PDF Generator ]   [ Excel Storage ]
        ↓                  ↓
     PDF File         Excel File
```

## 🧠 **B. Business Logic**

📁 `logic/bill_logic.py`

### Kaam:

* Total calculate karna
* Tax add karna
* Final amount

```python
def calculate_total(items):
    total = 0
    for item in items:
        total += item['qty'] * item['rate']
    return total
```
